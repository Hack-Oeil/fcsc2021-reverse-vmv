# FCSC 2021 VMV

Le flag est au format <code>FCSC{**chaine**}</code>, où **chaine** est la chaine de caractères 
à passer en argument de vmv qui affiche le message de réussite.


Fichier : 
- [skippy.py](skippy.py)
